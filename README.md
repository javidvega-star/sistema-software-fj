# sistema-software-fj

## Descripción
Sistema orientado a objetos desarrollado en Python para la gestión de clientes, servicios y reservas.

## Características
- Programación orientada a objetos
- Herencia
- Polimorfismo
- Encapsulación
- Manejo avanzado de excepciones
- Registro de logs
- Sistema robusto sin bases de datos

## Servicios implementados
- Reserva de salas
- Alquiler de equipos
- Asesorías especializadas

## Tecnologías utilizadas
- Python 3
- Visual Studio Code
- GitHub
